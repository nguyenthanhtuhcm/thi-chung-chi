# Exam Readiness Checklist

## AIP-C01 Readiness

Bạn sẵn sàng thi AIP-C01 khi:

- [ ] Đọc và tóm tắt được exam guide.
- [ ] Giải thích được 5 domain và trọng số.
- [ ] Hoàn thành RAG MVP.
- [ ] Hoàn thành agent có tool.
- [ ] Thêm guardrails và logging.
- [ ] Có evaluation set ít nhất 30 câu.
- [ ] Practice exam đạt 80-85% hai lần liên tiếp.
- [ ] Lỗi sai lặp lại đã được sửa.
- [ ] Hiểu khi nào dùng Bedrock Knowledge Bases, agents, Lambda orchestration.
- [ ] Hiểu IAM least privilege, encryption, PII và prompt injection risk.
- [ ] Hiểu cost/performance tradeoff: model selection, caching, token size, latency.

## MLA-C01 Readiness

Bạn sẵn sàng thi MLA-C01 khi:

- [ ] Đọc và tóm tắt được exam guide.
- [ ] Giải thích được 4 domain và trọng số.
- [ ] Làm được data preparation lab.
- [ ] Train được model trên SageMaker hoặc notebook.
- [ ] Hiểu metrics cho classification và regression.
- [ ] Deploy được endpoint hoặc batch transform.
- [ ] Hiểu model registry, pipeline, CI/CD concept.
- [ ] Hiểu CloudWatch monitoring và troubleshooting.
- [ ] Hiểu security: IAM, encryption, data protection.
- [ ] Practice exam đạt 80%+ ổn định.

## Quy Tắc 48 Giờ Trước Khi Thi

Nên làm:

- Đọc lại exam guide.
- Đọc mistakes log.
- Làm 1 mock exam nhẹ.
- Ngủ đủ.
- Chuẩn bị giấy tờ, thiết bị, Pearson VUE/test center.

Không nên làm:

- Học dịch vụ mới.
- Đổi nguồn học.
- Làm quá nhiều đề liên tục.
- Thức khuya.
