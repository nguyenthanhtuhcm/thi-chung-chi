# Thiết Kế AWS AI Developer Roadmap

## Mục Đích

Tạo một bộ tài liệu học tập có cấu trúc trong 10 tuần cho AI developer chuẩn bị thi AWS Certified Generative AI Developer - Professional (AIP-C01) trước, sau đó học tiếp AWS Certified Machine Learning Engineer - Associate (MLA-C01).

## Cách Tiếp Cận Đã Chọn

Dùng bộ tài liệu dạng module thay vì một file lớn. Cách này giúp roadmap dễ cập nhật, dễ review và dễ dùng trong từng ngày học.

## Kiến Trúc Tài Liệu

Bộ tài liệu nằm trong `aws-ai-developer-roadmap/` và gồm:

- Tổng quan và chiến lược chứng chỉ.
- Kiến trúc kiến thức.
- Kiến trúc capstone project.
- Kế hoạch học theo tuần.
- Template học hằng ngày.
- Lab deliverables.
- Checklist sẵn sàng thi.
- Danh sách tài nguyên.
- Template ghi chú cho exam notes, mistakes và practice scores.

## Capstone Project

Project là AWS GenAI Knowledge Assistant. Project bao gồm:

- Lưu trữ tài liệu trên S3.
- Ingestion, chunking, embeddings.
- RAG retrieval.
- Sinh câu trả lời bằng Amazon Bedrock.
- Tích hợp Lambda/API.
- Agent tools.
- Guardrails.
- Observability bằng CloudWatch.
- Phần mở rộng SageMaker cho MLA-C01.

## Tiêu Chí Hoàn Thành

Bộ tài liệu được xem là hoàn thành khi người học có thể:

- Theo dõi kế hoạch học theo ngày/tuần trong 10 tuần.
- Xây capstone project từng bước.
- Theo dõi lỗi sai và điểm mock exam.
- Quyết định thời điểm đăng ký thi AIP-C01 và MLA-C01 bằng checklist sẵn sàng thi rõ ràng.
